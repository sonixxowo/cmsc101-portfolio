# CMSC 101 Learning Portfolio — Elmerson Cruz

Monochrome, clean, tabbed website for weekly logs and executive summary.
Author: Elmerson Cruz, BSA‑1, University of the Philippines Cebu
Professor: Demelo Lao

## Features
- Monochrome light/dark theme with localStorage persistence
- Tabbed weeks with keyboard navigation
- Global content search
- Print to PDF with print styles
- Accessible, no external JS frameworks

## Develop
No build needed. Open `index.html` in a browser.

## Deploy
### GitHub Pages
1. Create a repo (e.g., `cmsc101-portfolio`).
2. Add the files in this folder.
3. Settings → Pages → “Deploy from a branch” → select `main`.
4. Your site will appear at `https://<username>.github.io/cmsc101-portfolio/`.

### Netlify / Vercel
- Drag and drop the folder into Netlify, or import the repo into Vercel.

## Customize
- Update `site.webmanifest` name/short_name if you change the title.
- Replace `favicon.svg` if you want a different glyph (keep it monochrome for consistency).
